\section*{Sample space}
A \textit{sample space} is a pair $(S,P)$ where $S$ is a finite, nonempty set and $P$ is a function $P:S\to\mathbb{R}$ such that $P(s)\geq 0$ for all $s\in S$ and
\[\sum_{s\in S}{P(s)}=1.\]
